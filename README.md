# Android Plugins
### 1. **Android Application Plugin**

- **ID**: `com.android.application`

- Location

  - Location: [https://mvnrepository.com/artifact/com.android.application/com.android.application.gradle.plugin](https://mvnrepository.com/artifact/com.android.application/com.android.application.gradle.plugin)
- **Purpose**: The core plugin used for Android application projects. It enables building and packaging of Android apps.

- **Usage**:

    ```groovy
    plugins {
        id 'com.android.application'
    }
    ```


### 2. **Android Library Plugin**

- **ID**: `com.android.library`

- Location

  - Location: [https://mvnrepository.com/artifact/com.android.library/com.android.library.gradle.plugin](https://mvnrepository.com/artifact/com.android.library/com.android.library.gradle.plugin)
- **Purpose**: Used to build Android libraries (AARs), allowing code and resources to be shared across multiple Android projects.

- **Usage**:

    ```groovy
    plugins {
        id 'com.android.library'
    }
    ```


### 3. **Kotlin Android Plugin**

- **ID**: `kotlin-android`

- Location

  - Location: [https://mvnrepository.com/artifact/org.jetbrains.kotlin/kotlin-android-extensions-runtime](https://mvnrepository.com/artifact/org.jetbrains.kotlin/kotlin-android-extensions-runtime)
  - [https://plugins.gradle.org/plugin/org.jetbrains.kotlin.android](https://plugins.gradle.org/plugin/org.jetbrains.kotlin.android)
- **Purpose**: Adds support for Kotlin in Android projects. This is essential for Kotlin-based Android development.

- **Usage**:

    ```kotlin
    plugins {
      id("org.jetbrains.kotlin.android") version "2.1.0-Beta1"
    }
    ```

### 4. **Kotlin Android Extensions Plugin**

- **ID**: `kotlin-android-extensions`

- Location:

  - [https://mvnrepository.com/artifact/org.jetbrains.kotlin/kotlin-android-extensions-runtime](https://mvnrepository.com/artifact/org.jetbrains.kotlin/kotlin-android-extensions-runtime)
  - [https://plugins.gradle.org/plugin/org.jetbrains.kotlin.android.extensions](https://plugins.gradle.org/plugin/org.jetbrains.kotlin.android.extensions)
- **Purpose**: Provides features like synthetic view access, but this is deprecated. Use `ViewBinding` or `DataBinding` for view binding.

- **Usage**:

    ```groovy
    plugins {
        id 'kotlin-android-extensions'
    }
    ```


### 5. **Kotlin Serialization Plugin**

- **ID**: `kotlinx-serialization`

- **Purpose**: Adds support for Kotlin Serialization, allowing easy serialization and deserialization of Kotlin data classes.

- **Usage**:

    ```groovy
    plugins {
        id 'kotlinx-serialization'
    }
    ```

- Location

  - Github: [https://github.com/Kotlin/kotlinx.serialization](https://github.com/Kotlin/kotlinx.serialization)
  - [https://mvnrepository.com/artifact/org.jetbrains.kotlin.plugin.serialization/org.jetbrains.kotlin.plugin.serialization.gradle.plugin](https://mvnrepository.com/artifact/org.jetbrains.kotlin.plugin.serialization/org.jetbrains.kotlin.plugin.serialization.gradle.plugin)
  - Kotlin compiler plugin for kotlinx.serialization library
  - [https://github.com/Kotlin/kotlinx.serialization?tab=readme-ov-file#setup](https://github.com/Kotlin/kotlinx.serialization?tab=readme-ov-file#setup)
  - [https://plugins.gradle.org/plugin/org.jetbrains.kotlin.plugin.serialization](https://plugins.gradle.org/plugin/org.jetbrains.kotlin.plugin.serialization)
  - [https://kotlinlang.org/docs/serialization.html#add-plugins-and-dependencies](https://kotlinlang.org/docs/serialization.html#add-plugins-and-dependencies)

```kotlin
plugins {
    kotlin("jvm") version "2.0.20" // or kotlin("multiplatform") or any other kotlin plugin
    kotlin("plugin.serialization") version "2.0.20"
}
```

```groovy
plugins {
    id 'org.jetbrains.kotlin.multiplatform' version '2.0.20'
    id 'org.jetbrains.kotlin.plugin.serialization' version '2.0.20'
}
```
